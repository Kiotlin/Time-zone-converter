## 使用指南
提供两种方式使用conv-ti

#### 直接访问
[conv-ti](https://kiotlin.github.io/conv-ti/)  
使用方法见[网站](https://kiotlin.github.io/conv-ti/)主页`# Manual` 区域

#### 部署到本地
```
git clone https://github.com/Kiotlin/conv-ti.git
cd conv-ti
npm install
npm run start
```
